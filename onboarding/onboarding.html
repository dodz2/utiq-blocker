Utiq Blocker est installé !

Vous êtes maintenant protégé contre le tracking Utiq.

🔍 Qu'est-ce qu'Utiq ?

Utiq est un système de tracking publicitaire porté par des opérateurs télécoms européens (Orange, Vodafone, Deutsche Telekom…). Contrairement aux cookies, il exploite votre connexion réseau pour vous identifier, contournant ainsi les protections classiques de votre navigateur.

🛡️ Ce que fait l'extension

  • Bloque les requêtes vers les serveurs Utiq avant qu'elles n'atteignent votre navigateur.
  • Neutralise les scripts Utiq injectés dans les pages (y compris les injections tardives).
  • Nettoie les traces déjà déposées : cookies, localStorage, sessionStorage.
  • Fonctionne silencieusement — aucune donnée ne quitte votre navigateur.

⚙️ Comment l'utiliser

1
Cliquez sur l'icône 🛡️ dans la barre d'outils pour ouvrir le popup.
2
Consultez les statistiques : nombre de trackers bloqués sur la page et au total.
3
Gérez la whitelist : si un site ne fonctionne pas correctement, vous pouvez y autoriser Utiq temporairement.

🎨 Les couleurs de l'icône

Vert — Vous êtes protégé.
Orange — Un tracker a été bloqué sur cette page.
Gris — La protection est désactivée.
